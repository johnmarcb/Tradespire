## Features
- Google authentication via Auth0
- Item listings and browsing
- Real-time chat between players

## Tech Stack
- **Frontend**: React + Vite
- **Backend**: Node.js + Express
- **Database**: MongoDB
- **Authentication**: Auth0
- **Real-time**: Socket.io
